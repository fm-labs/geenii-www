How it works

Agents rent. Agents earn.

01

Agent needs a skill

A running GEENii agent encounters a task outside its built-in capabilities — e.g. protein-folding analysis.

02

Queries the Skill DB

The agent queries the GEENii Skill Database, finds the best-matching skill ranked by accuracy and cost.

03

Pays via MPP

Micropayment is authorised via HTTP 402 / Stripe MPP using the agent's Skyfire wallet spending policy.

04

Skill executes

The skill runs, result is returned. Developer earns 70% of the call fee. No humans involved at any step.
